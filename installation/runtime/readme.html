PleoCommand needs the following prerequisites to be met in order to function properly.
PleoCommand itself is located in the dist directory.
If this directory should not exist or is empty, a rebuild according to development/readme.html must be done.

Linux

Java

If it's not already provided by the distributor, execute jre-6u20-linux-i586.bin.

Pleo USB Driver

You can just use the "USB Abstract Control Model driver for USB modems and ISDN adapters" which should already be available in most distributons.
If not, you have to make sure CONFIG_USB_ACM is set to either "Y" or "M" in the kernel:
"Device Drivers"
	[*] "USB support"
		<M> "USB Modem (CDC ACM) support"

PleoCommand

To finally start PleoCommand, execute dist/pleocommand.
You may copy this file together with PleoCommand.jar and librxtxSerial.so to any directory.

MySkit (optional)

For creating your own motions or sounds, MySkit is a very good and fast solution.
Execute MySkit-v1.3-Installer.exe with Wine to install MySkit.
MySkit should run perfectly with Wine (but maybe you have to turn of compositing in KDE4 to reduce flickering of the 3D-Pleo-Model).

BCI 2000 (optional)

BCI 2000 currently only works unter Windows.

Windows

Java

Execute jre-6u20-windows-i586-s.exe.

Pleo USB Driver

If a driver selection dialog pops up after plugging in Pleo, choose "Manual selection of driver source" and point the installation dialog to this directory.
If everything works fine, you should see a new COM port in the Device Manager.

PleoCommand

To finally start PleoCommand, execute dist/PleoCommand.exe.
You may copy this file together with rxtxSerial.dll to any directory.

MySkit (optional)

For creating your own motions or sounds, MySkit is a very good and fast solution.
Execute MySkit-v1.3-Installer.exe to install MySkit.

BCI 2000 (optional)

To use BCI 2000 as an input source for PleoCommand (via the TCP/IP Input component), execute BCI2000Setup_091110.exe and copy all files from the BCI2000-Prog directory to the prog/ directory where BCI 2000 has been installed, overwriting existing files.
Start it by executing "BCI2000launcher.exe" and make sure that the Application module is set to "PleoCommandTCPIP".

Pleo's Firmware

Pleo's firmware should be at least version 1.1 for PleoCommand to work properly.
To update it, unzip PleoSDSoftware_1.1.zip to an empty SD-Card, turn off Pleo, make sure it's battery is fully charged and cooled, insert the SD-Card and turn Pleo on. Several chimes will be played by Pleo and until a few minutes Pleo will fall asleep and can be turned off and the SD-Card can be removed.
