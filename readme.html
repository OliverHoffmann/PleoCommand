src All source files. This are the only one needed to generate all files in all the other directories.
bin Contains temporary files creating during building. Can safely be deleted.
dist Contains all finally created files, ready for distribution. Can be recreated from src/ if deleted.
ext/rt Additional external dependencies which may be required to execute files in dist/. Can be recreated from src/ if deleted.
ext/dev Additional external dependencies which may be required to (re)build, modify or extend files in src/. Can be recreated from src/ if deleted.
lib External libraries required for building. TODO: remove this dir
