Howto: Neue Komponenten erstellen

  1. Eine bereits vorhandene Komponente als Vorlage verwenden bzw. neue Klasse mit Input, Converter oder Output Klasse als Superklasse erstellen.
    Input-Komponenten müssen in pleocmd.pipe.in liegen
    Konverter-Komponenten müssen in pleocmd.pipe.cvt liegen
    Output-Komponenten müssen in pleocmd.pipe.out liegen
  2. Abstrakte Methoden anpassen bzw. implementieren
  3. public static String help(final HelpKind kind) anpassen bzw. hinzufügen
  4. Optional: Hilfedatei mit Klassenname plus ".html" in src/pleocmd/itfc/gui/help erstellen
  5. Optional: Bild (etwa 250 x 150 Pixel) für Konfigurations-Dialog mit Klassenname plus "-cfg.png" in src/pleocmd/itfc/gui/icons erstellen
  6. Optional: Icon (maximal 32 x 32 Pixel) für Board mit Klassenname plus "-icon.png" in src/pleocmd/itfc/gui/icons erstellen
  7. Entweder: PleoCommand neu bauen (ant dist ausführen)
    Oder: Unter Beibehaltung obiger Ordnerstruktur alle neu erstellten Dateien in ein .jar Archiv schreiben und dieses im dist/ Ordner ablegen für automatisches Einbinden, ohne dass PleoCommand neu gebaut werden muss.
