Pipe-Konfigurations-Dialog

Bereiche

Die Pipe-Konfiguration basiert auf einem Zeichenbrett (Board), welches vertikal in drei Bereiche unterteilt ist:
Links ist der Bereich für Eingabe-Komponenten. Sie werden bei Ausführen der Pipe von oben nach unten der Reihe nach abgearbeitet. Wurde der letzte ausgeführt, wird die Pipe beendet.
Im mittleren Bereich befinden sich die Konverter und im rechten Bereich die Ausgabe-Komponenten.

Interaktion mit der Maus


Komponenten haben einen inneren und äußeren Bereich, außerdem können sie Buttons enthalten. Zwischen Komponenten können Verbindungen erstellt werden.
Durch Verweilen mit der Maus auf einer Komponente wird eine Übersicht über die Konfiguration der Komponente sowie statistische Informationen angezeigt.
Durch Links-Klicken auf den inneren Bereich und Ziehen der Maus kann eine Komponente verschoben werden. Der dunkelgrün hervorgehobene Bereich auf dem Board signalisiert den Bereich, in welchem eine Komponente verschoben werden kann, ohne die interne Ordnung bei Ausführung der Pipe zu verändern.
Durch Links-Klicken auf den äußeren Bereich und Ziehen der Maus kann eine (weitere) Verbindung von dieser Komponente zu einer anderen erstellt werden. Die neue Verbindung wird erst bei Loslassen der Maus über der gewünschten Zielkomponente erstellt und nur unter der Voraussetzung, das eine Verbindung zwischen diesen Komponenten erlaubt ist und nicht bereits existiert.
Durch Links-Klicken auf eine Verbindung und Ziehen der Maus kann die Verbindung zu einem anderen Ziel verschoben werden. Durch Loslassen der Verbindung über einem ungültigen Ziel wird die Verbindung entfernt.
Durch einfaches Links-Klicken auf einen Button kann der Konfigurationsdialog aufgerufen oder die Darstellung eines Diagramms aktiviert werden.

Kontextmenü

Das Kontextmenü wird durch Klicken mit der rechten Maustaste auf das Bord oder eine Komponente bzw. Verbindung auf dem Board aufgerufen:
Mit "Add ..." können zu diesem Bereich passende Komponenten zum Board hinzugefügt werden.
Mit "Replace ..." kann die aktuell markierte Komponente durch eine andere ersetzt werden.
Mit "Configure This PipePart" wird der Konfigurationsdialog der Komponente aufgerufen (kann auch durch Doppelklick auf eine Komponente erreicht werden).
Mit "Delete This PipePart" wird die Komponente vom Board entfernt.
Mit "Delete Connections ..." werden alle ausgehenden Verbindungen der Komponente vom Board entfernt.
Mit "Delete This Connection" bei Klick auf eine einzelne Verbindung kann diese entfernt werden (kann auch durch Ziehen der Verbindung mit der Maus ins Nichts erreicht werden).
Mit "Clear The Board" werden alle Komponenten und deren Verbindungen vom Board entfernt.
Mit "Auto-Layout The Board" werden alle Komponenten und deren Verbindungen so umsortiert, dass möglichst wenig Überkreuzungen entstehen.
