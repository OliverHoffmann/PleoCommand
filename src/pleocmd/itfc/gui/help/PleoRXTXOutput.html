Pleo RXTX Output

Baut eine Verbindung über einen COM Port zu Pleo auf und sendet Anweisungen an Pleo.
Erwartet Datenblöcke welche mit "PMC" beginnen, zum Beispiel "PMC|motion play happy".
Eine Übersicht über alle möglichen Befehle bietet das Dokument "Ugobe Pleo Monitor", welches Bestandteil des PleoDevelopmentKits ist (dieses befindet sich im Ordner "ext/dev" von PleoCommand).
Durch Eingabe von "PMC|help" kann auch direkt eine Liste aller möglichen Kommandos abgefragt werden.

Solange eine Pipe aktiv ist, welche diese Komponente enthält, wird in der Statusanzeige des Hauptfensters der Zustand der Batterie des angeschlossenen Pleo angezeigt.

Die Firmware von Pleo sollte mindestens auf dem Stand von Version 1.1 sein.
Unter Windows muss zum Zugriff auf Pleo der Pleo-USB-Treiber installiert sind.
Eine passende Kopie des Treibers und der Firmware befindet sich im Ordner "ext/rt" von PleoCommand.

Unter Linux muss lediglich im Kernel das Modul "USB Abstract Control Model driver for USB modems and ISDN adapters" verfügbar sein.
Um einen für die von PleoCommand verwendete Bibliothek passenden Symbolnamen zu erstellen, sollte eine Datei mit dem Namen "/etc/udev/rules.d/10-pleo.rules" und folgendem Inhalt: 
KERNEL=="ttyACM*", SUBSYSTEM=="tty", ATTRS{idVendor}=="6962", SYMLINK+="ttyS9"
erstellt werden.
