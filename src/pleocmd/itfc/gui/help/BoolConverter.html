Bool Converter

Sendet für jedes empfangene Datenpaket vom Typ Single-Float ein Datenpaket vom Typ Single-Boolean.
Der enthaltene Wert ist wahr(1), falls die festgelegte Vergleichsoperation gültig wird, ansonsten falsch(0).
Beispiel für >= 4: 
  Eingabe: 2 1 3 5 0 6 5 4 3 2
  Ausgabe: 0 0 0 1 0 1 0 0 0 0
Dies ist lediglich ein sehr einfacher Converter:
Komplexere Ausdrücke als einfache Vergleiche lassen sich mit Hilfe des Expression Converter durchführen.
Eine Bereichsprüfung kann auf flexiblere Weise mit Hilfe des Threshold Converter erreicht werden.
