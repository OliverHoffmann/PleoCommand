Threshold Converter

Erwartet eingehende Daten vom Typ Single-Float. Diese werden je nach Konfiguration in einen Wahrheitswert oder einen von zwei möglichen Fließkomma-Werten konvertiert. Bei Wahrheitswerten als Ausgabe wird Wahr ausgegeben, wenn der Eingangswert über dem Grenzwert liegt und Falsch, falls er darunter liegt. Sprünge über den Grenzwert von innerhalb [Threshold - Marginal, Threshold + Marginal] werden jedoch ignoriert und erzeugen somit keinen Wechsel des Wahrheitswertes. Sollen Fließkommazahlen statt Wahrheitswerte ausgegeben werden, so wird Value For Below respektive Value For Above verwendet. Ist letzterer Null, so wird stattdessen der aktuelle Eingangs-Wert ausgegeben.
