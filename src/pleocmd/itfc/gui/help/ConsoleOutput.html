Console Output

Schreibt Datenpakete auf die Konsole in einem der folgenden Formate:
Ascii Alle Datenpakete werden in Textform geschrieben
Binary Alle Datenpakete werden in Binärform geschrieben
AsciiOriginal Schreibt nur das erste Datenpaket aus einer Sequenz von Datenpaketen mit identischer Quelle (in Textform)
BinaryOriginal Schreibt nur das erste Datenpaket aus einer Sequenz von Datenpaketen mit identischer Quelle (in Binärform)
PleoMonitorCommands Schreibt lediglich Kommandos, welche mit "PMC" beginnen (in Textform)
Im graphischen Benutzermodus erscheint die Ausgabe im Log-Bereich des Hauptfensters, im Kommandozeilenmodus werden die Datenpakete auf die Standardausgabe geschrieben. Die Darstellung binärer Datenpakete im graphischen Benutzermodus wird dabei durch Umwandlung in hexadezimale Form simuliert.
