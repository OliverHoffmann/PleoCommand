File Output

Schreibt Datenblöcke in eine Datei in einem der folgenden Formate:
Ascii Alle Datenblöcke werden in Textform geschrieben
Binary Alle Datenblöcke werden in Binärform geschrieben
AsciiOriginal Schreibt nur das erste Datenblock aus einer Sequenz von Datenblöcken mit identischer Quelle (in Textform)
BinaryOriginal Schreibt nur das erste Datenblock aus einer Sequenz von Datenblöcken mit identischer Quelle (in Binärform)
PleoMonitorCommands Schreibt lediglich Kommandos, welche mit "PMC" beginnen (in Textform)
Für Datenblöcke in Binärform sollte die Datei eine Endung ".pbd" besitzen, für TextForm Datenblöcke wird ".pad" empfohlen.

Siehe auch Aufbau eines binären Datenblocks.
Siehe auch Aufbau eines ASCII Datenblocks.
