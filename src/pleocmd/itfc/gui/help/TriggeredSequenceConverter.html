Triggered Sequence Converter

Erwartet Datenblöcke mit einer Zeichenfolgen als erstes Feld und einem beliebigen Wert als zweites Feld. Stimmt der Inhalt des ersten Feldes mit einem festlegbaren Präfix-Wort überein, so wird ein Kommando (oder eine Liste an Kommandos) basierend auf dem Wert des zweiten Feldes ausgegeben.
Welcher Wert hierbei zu welcher Kommando-Liste führt, lässt über die Konfiguration festlegen. Die Kommandos können einen beliebigen Inhalt aufweisen, theoretisch wären auch vordefinierte Zahlen(folgen) als Ausgabe möglich.
Zur Eingabe einer neuen Abbildung wird zuerst auf Add geklickt, anschließend der Name des Wertes (zweites Feld des Eingang-Datenblocks) eingegeben. Anschließend wird im darunterliegenden Eingabebereich die Liste an Ziel-Kommandos eingegeben.
Zu Testzwecken können diese mit Play Selected bzw. Play All auch direkt an den Pleo übergeben und abgespielt werden.
Mit Add From File bzw. Copy From Input History können Kommandos aus einer externen Datei bzw. den zuletzt im Hauptfenster eingegebenen Kommandos übernommen werden.
