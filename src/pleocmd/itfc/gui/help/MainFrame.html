Haupt-Dialog

Der Haupt-Dialog ist vertikal in drei Bereiche unterteilt. Die Größe des Aus- und Eingabebereichs kann beliebig angepasst werden.

Pipe Übersicht

Eine verkleinerte Übersicht über die aktuell gültige Pipe.
Kann durch Klicken auf "Modify" bearbeitet werden, außerdem können Pipe-Konfigurationen in externe Dateien gespeichert und von dort wieder geladen werden. Durch Speichern einer Pipe in eine Datei wird der verwendete Dateiname automatisch zum Namen der Pipe. Die aktuell gültige Pipe wird bei Beenden des Programms automatisch in einer separaten (nicht in der zuletzt verwendeten) Datei gesichert und bei erneutem Programmstart wiederhergestellt.

Ausgabe

Hier erscheinen sämtliche Hinweise und Fehlermeldungen. Außerdem werden sämtliche von der ConsoleOutput-Komponente empfangenen Datenblöcke hier ausgegeben.
Durch Klicken auf "Show detailed Log" werden zusätzlich Debug-Meldungen angezeigt. Dies kann die Ausführungsgeschwindigkeit des Programms jedoch stark reduzieren. Mit dem "Start"-Button wird die Pipe gestartet, mit dem "Abort"-Button kann sie gestoppt werden.

Eingabe

Die Eingabe ist mit der ConsoleInput-Komponente verbunden. Existiert keine solche Komponente in der aktuellen Pipe, so ist auch keine Eingabe möglich.
Alle jemals eingegebenen Kommandos werden in der History gespeichert und können durch einfaches Klicken in die Eingabezeile geholt oder durch doppeltes Klicken direkt ausgeführt werden. Liegt der Fokus auf der Eingabezeile, kann die History durch Cursor hoch und runter ebenfalls durchlaufen werden.
Durch Drücken von Strg+R wird der erste Eintrag in der History, welcher die aktuelle Eingabe enthält, gesucht und in die Eingabezeile übernommen.
"Send" sendet die aktuelle Eingabe an die ConsoleInput-Komponente und entspricht dem Drücken von Return oder Doppelklicken in die History.
Mit "SendEOS" wird die Eingabe terminiert. Befindet sich in der Pipe nach dem ConsoleInput noch eine weitere Input-Komponente, so wird diese nun ausgeführt. Existiert keine weitere Komponente, wird die Pipe beendet.
Durch Klicken auf "Read From..." kann eine externe Datei eingelesen und deren Inhalt ausgeführt werden, als ob er Zeile für Zeile eingegeben worden wäre.
