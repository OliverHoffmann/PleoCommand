TCP/IP Input

Wartet auf eine TCP/IP Verbindung an einem definerbaren Port und gibt darüber erhaltene Datenblöcke aus.
Ein Abbruch der Verbindung wird als Ende der Datensequenz gewertet.
Datenblöcke werden in Binärform erwartet und als Multi-Float-Datenblöcke ausgegeben.
