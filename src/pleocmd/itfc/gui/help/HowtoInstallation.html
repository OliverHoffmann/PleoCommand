Howto: Installation

Der Unterordner dist/ enthält alle relevanten Dateien von PleoCommand. Der Inhalt dieses Ordners kann auf einen beliebigen Rechner kopiert werden, um PleoCommand dort auszuführen. Zusätzlich ist für die Ausführung (Messen, Erstellen eigener Motions, ...) der Inhalt des Ordners ext/rt sowie für fortgeschrittene Benutzung (Bauen eigener Verhaltensmuster innerhalb Pleo, ...) ext/dev sinnvoll.
Einzelne Schritte:
  1. Ordner dist/ und ext/ auf Zielrechner kopieren
  2. Java installieren
  3. Aus ext/rt: BCI2000-setup.zip entpacken und installieren
  4. Aus ext/rt/BCI2000-Prog: Inhalt in bin-Ordner der BCI 2000-Installation kopieren
  5. Pleo einstecken
  6. Windows: Bei Frage nach Treiber den Inhalt von Ordner ext/rt verwenden
    Linux: Pleo wird automatisch als USB Modem (CDC ACM) erkannt
  7. MySkit aus ext/rt installieren
Näheres in der readme.html im Hauptordner von PleoCommand.
