Internal Commands

Führt bestimmte Reaktionen aus, wenn Datenpakete mit "SC" als erstes Feld empfangen werden:
SLEEP Blockiert die Ausgabe nachfolgender Datenpakete (egal welchen Formats) um eine bestimmte Zeit (in ms), gegeben durch ein weiteres Feld
HELP Gibt die integrierte Kurzhilfe aller vorhandenen Inputs, Converter und Outputs aus oder, bei Vorhandensein weiterer Felder nur der aufgelisteten Komponenten
ECHO Gibt den im folgenden Feld enthaltenen Text auf der Konsole aus, wobei bestimmte Zeichenfolgen eine besondere Bedeutung besitzen und bei Vorkommen innerhalb des Textes ersetzt werden:
$ELAPSED Zeit in ms seit Start der Pipe
$PACKETS Anzahl der eingelesenen Datenpakete
$STATS Allgemeine Statistik

Beispiele:

SC|SLEEP|500
SC|HELP
SC|HELP|FileInput|StaticInput
ECHO|Pipe is running since $elapsed ms
